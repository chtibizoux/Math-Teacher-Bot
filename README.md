# Math Teacher Bot
A discord bot for Maths.

Commandes:

1. `)calculator` Aide de la calculatrice
2. `)aide MonProblème` Affiche une aide
3. `)training MonExo` Affiche un Exercice de maths
4. `)table f 0 10 1` Affiche le tableau de la fonction `f(x)` avec x de 0 à 10 et un pas de 1
5. `)graph f g` Trace le graphique des fonctions `f` et `g`
6. `)functions` Affiche toutes les fonctions
